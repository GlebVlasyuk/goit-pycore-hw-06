# goit-pycore-hw-06

Python Core Homework #6: an address book management system implemented in OOP style.

What was implemented:
- Implemented classes: `Field`, `Name`, `Phone`, `Record`, `AddressBook`.
- Added phone validation in `Phone` (exactly 10 digits).
- Implemented methods in `Record`: `add_phone`, `remove_phone`, `edit_phone`, `find_phone`.
- Implemented methods in `AddressBook`: `add_record`, `find`, `delete`.

Project structure:
- `address_book.py` — core logic and classes.
- `main.py` — usage example (create records, search, edit, delete).
